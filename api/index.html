API Reference

This is a comprehensive reference of the libraries that make up Gir.Core. Use the sidebar on the left to navigate to a specific library.

Note that the documentation here is in large part automatically generated from gtk-doc comments in source code. This means that certain parts of the project may not be documented, or documentation might not work correctly. If in doubt, always defer to the upstream API References. These can be found at the GNOME Developer Centre.

Important

Gir.Core is currently in Alpha. API may change at any time and Documentation is largely incomplete. If you'd like to help, please get in touch!

Libraries

Main Libraries

These are the libraries you will likely use often.

  • Gtk - GUI Toolkit
  • Pango - Text Layout & Display
  • Cairo - Graphics
  • Gdk - OS Integration for Gtk
  • GdkPixbuf - Image Handling
  • GObject - Object oriented type system. Forms the basis for all other libraries.

Optional Libraries

The following libraries are a core part of the Gtk stack, but provide functionality similar to that of the .NET Runtime (e.g. File I/O). Using these libraries directly is optional in most cases; they are provided for completeness.

  • GLib - System functions
  • Gio - Cross Platform Input/Output.
